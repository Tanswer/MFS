----------
### Programming Model
 1. I/O 多路服用（epoll   ET）
 2. Non-blocking I/O      、解析读取到的数据时维护一个状态机
 3. 线程池

### How To Run
please make sure you have [cmake](https://cmake.org/) installed.
```
git clone git@github.com:Tanswer/Xserver.git

cd Xserver

./autorun

```

### Complete

 1. http persistent connection
 2. serve static(.html)
 3. GET / POST
 4. some HTTP/1.1 features
